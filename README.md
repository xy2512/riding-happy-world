# 🚴 骑乐星球 — 游戏化骑行激励产品

> 骑行记录 · 成长成就 · 真实奖励

一款面向城市通勤族和骑行爱好者的游戏化骑行 Web 产品，通过地图寻宝、宝箱抽奖、成长等级、轻社交等机制解决骑行枯燥和留存低的问题。

## 🎮 核心玩法

- **🗺️ 地图寻宝** — 2D地图随机散落金币/宝箱/钻石，骑行经过自动拾取
- **🎁 宝箱抽奖** — 骑行≥3km获得宝箱，随机获取骑行币/装扮/优惠券
- **⬆️ 等级成长** — 10级等级体系 + 9枚成就徽章 + 连续打卡
- **👋 轻社交** — 「擦肩而过」展示同路骑友，无好友压力
- **🏍️ 数字装扮** — 2D单车个性化装扮（车漆/铃铛/挂件/头盔）

## 📊 核心指标

- **北极星指标**：周活跃骑行里程
- **增长指标**：DAU、新用户注册
- **留存指标**：次日/7日/30日留存率
- **游戏化指标**：彩蛋拾取率(≥80%)、宝箱开启率(≥90%)

## 🛠️ 技术栈

- 原型：纯 HTML + Tailwind CSS + Vanilla JS（单文件，零依赖）
- 流程图：Mermaid.js
- 设计风格：emerald 绿色主题，移动端优先（max-width: 430px）
